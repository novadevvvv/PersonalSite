About

Transparency

How SkidChecker evaluates Minecraft mods

SkidChecker is a static analysis tool designed for rapid Minecraft mod triage. It does not execute uploaded jars. Instead, it decompiles the archive, inspects the resulting source, and scores detections using severity-weighted rules and confidence values.

Static only

Submitted jars are decompiled and reviewed as source text. No runtime execution or sandbox detonation is performed.

Browser-mode support

On GitHub Pages, SkidChecker can inspect jar contents directly in the browser without a backend. A hosted API remains optional for deeper server-side analysis.

Confidence-scored rules

Each check includes a confidence level and rationale so results can be interpreted with appropriate caution.

Analyst review remains important

SkidChecker is intended to support review and prioritization, not replace manual inspection for high-risk samples.
